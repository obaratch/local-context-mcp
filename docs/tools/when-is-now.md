# when-is-now

## 目的
- AI とユーザが同じ環境にいる前提で、現在日時の文脈を共有するための基本機能。

## 概要
- `when-is-now` は、MCP サーバ実行環境における現在日時を返す。
- この MCP サーバはユーザと同じ場所にある想定のため、返却する日時のタイムゾーンはサーバのローカルタイムゾーンに従う。
- UTC 固定ではない。

## 入力
- なし

## 返却方針
- 現在日時を ISO 8601 形式の文字列で返す。
- タイムゾーンのオフセットを含める。
- 返却値に含まれるタイムゾーンは、サーバ実行環境のローカルタイムゾーンとする。

## 例
- `2026-03-18T15:21:00+09:00`
